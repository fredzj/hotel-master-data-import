# Hotel Master Data Import System

A robust Laravel application designed to integrate with multiple Property Management Systems (PMS) and serve as a central hub for managing hotel-related data with multi-tenant authentication and comprehensive CRUD functionality.

## üöÄ Features

- **Multi-PMS Integration**: Extensible adapter pattern for integrating with various Property Management Systems
- **Apaleo Integration**: Complete OAuth2 integration with Apaleo PMS
- **Multi-Tenant Architecture**: Role-based access control with Super Admin and Hotel Staff roles
- **Background Processing**: Queued jobs with database driver for shared hosting compatibility
- **Dynamic Dashboard**: Real-time import progress tracking and comprehensive statistics
- **Full CRUD Operations**: Complete management interface for all hotel entities
- **Shared Hosting Ready**: Optimized for standard shared hosting environments

### Core Technologies
- **Backend**: Laravel 12 (PHP 8.2+)
- **Database**: MySQL
- **Frontend**: Blade templates with Bootstrap CSS
- **Authentication**: Laravel UI with Spatie Laravel Permission
- **Queue System**: Database driver with Laravel Scheduler
